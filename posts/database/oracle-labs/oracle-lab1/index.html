Previous work#

在MacOS环境下配置Oracle并非一件易事，课程学习需要，我查阅博客，利用Docker进行环境配置。 首先在官网选择适合自己电脑环境的Docker版本下载。 浏览了一遍菜鸟教程中的Docker入门，对Docker有一个初步的了解。

  1. Docker 允许你在容器内运行应用程序， 使用 docker run 命令来在容器内运行一个应用程序。
$ docker run ubuntu:15.10 /bin/echo "Hello world"

各个参数解析：

  • docker: Docker 的二进制执行文件。
  • run: 与前面的 docker 组合来运行一个容器。
  • ubuntu:15.10 指定要运行的镜像，Docker 首先从本地主机上查找镜像是否存在，如果不存在，Docker 就会从镜像仓库 Docker Hub 下载公共镜像。
  • /bin/echo "Hello world": 在启动的容器里执行的命令。
  1. 我们通过 docker 的两个参数 -i -t，让 docker 运行的容器实现"对话"的能力： shell (base) ➜ ~ docker run -d -p 151:151 --name oracle store/oracle/database-enterprise:12.2.0.1-slim a0bb03a0f3807f2c3c8c10912302d1fab3588c1197748bd28304834926c0d1e5 (base) ➜ ~ docker exec -it oracle bash [oracle@a0bb03a0f380 /]$ 各个参数解析：
  2. -t: 在新容器内指定一个伪终端或终端。
  3. -i: 允许你对容器内的标准输入 (STDIN) 进行交互。

注意[oracle@a0bb03a0f380 /]$，此时我们已进入一个 oracle 系统的容器

Lab#

实验要求1#

启动oracle的服务，执行SQL*PLUS。

  • 输入：system/manager(密码可能有变化)；
  • 在SQL>状态下输入建立用户命令:create user username identified by password，其中username与password不要数字开头，用户名username中按规则加学号后3位作后缀。如：U_J101表示尾数为J101号的用户;
  • 给用户授权：grant resource,connect to username;
  • 连接用户：connect username/password
具体操作1#

在oracle容器内执行sqlplus / as sysdba

[oracle@a0bb03a0f380 /]$ sqlplus / as sysdba

建立用户，授权并连接：

create user c##u_d710 identified by lab1;
grant resource,connect,dba to c##u_d710;
conn c##u_d710/lab1

实验要求3#

建立学生信息表t_stud_d710(SNO primary key, SNAME, SEX (男，女，其他),TEL ,E-MAIL (基本格式判断), birthday >=’19990731’ ,MNO 班长学号 FK，MajorNo 专业编号 FK )，其中学号的第3，4位为MajorNo（专业编号）。

难点： 学号的第3，4位要求为MajorNo（专业编号），我们只能自己设计一个两位的专业编号

具体操作3#
create table t_stud_d710(
    sno varchar2(10),
    sname varchar2(32),
    sex varchar2(32),
    tel varchar2(32),
    email varchar2(32),
    birthday date,
    mno varchar2(10),
    majorno varchar2(2),
    constraint pk_stud_sno primary key(sno),
    constraint ck_stud_sex check(sex in('男','女','其它')),
    constraint ck_stud_email check(email like '%@%.%'),
    constraint ck_stud_birthday check(birthday>to_date('19990731','yyyymmdd')),   
    constraint ck_stud_majorno check(majorno in substr(sno,3,2)),    
    constraint fk_stud_mno foreign key(mno) references t_stud_d710(sno),
    constraint fk_stud_majorno foreign key(majorno) references t_major_d710(mno));

实验要求6#

为每个学生建立相关用户，实现权限控制，每个学生可查询自己的信息，

班长可查询本班所有学生信息； 为每个专业负责人建立用户，

每个专业负责人可查询本专业所有学生信息。

具体操作6#
--删除用户
select 'drop user u'||sno||';' from t_stud_d710;
--新建用户（为每个学生）
select 'create user u'||sno||' identified by p'||sno||';' from t_stud_d710;
--授予connect权利
select 'grant connect to u'||sno||';' from t_stud_d710;

--删除学生用户视图
select 'drop view vi'||sno||';' from t_stud_d710;
--为每个用户建立各自的视图
select 'create view vi'||sno||' as select * from t_stud_d710 where sno='||sno||';' from t_stud_d710;
--为每个用户授予查询视图的权利
select 'grant select on vi'||sno||' to u'||sno||';' from t_stud_d710;
--删除班长用户视图
select 'drop view vi'||sno||';' from t_stud_d710 where sno=mno;
select 'drop view vib'||sno||';' from t_stud_d710 where sno=mno;
--为班长建立班长视图
select 'create view vib'||sno||' as select * from t_stud_d710 where mno='||sno||';' from t_stud_d710 where sno=mno;
--为班长授予查询班长视图的权利
select 'grant select on vib'||sno||' to u'||sno||';' from t_stud_d710 where sno=mno;
--删除年级负责人用户视图
select 'drop view vi'||t_stud_d710.sno||';' from t_stud_d710,t_major_d710 where t_stud_d710.majorno=t_major_d710.mno and t_stud_d710.sname=t_major_d710.mdean;
select 'drop view vib'||t_stud_d710.sno||';' from t_stud_d710,t_major_d710 where t_stud_d710.majorno=t_major_d710.mno and t_stud_d710.sname=t_major_d710.mdean;
select 'drop view viz'||t_stud_d710.sno||';' from t_stud_d710,t_major_d710 where t_stud_d710.majorno=t_major_d710.mno and t_stud_d710.sname=t_major_d710.mdean;
--为年级负责人创建年级视图
select 'create view viz'||t_stud_d710.sno||' as select * from t_stud_d710 where majorno='||t_stud_d710.majorno||';' from t_stud_d710,t_major_d710 where t_stud_d710.majorno=t_major_d710.mno and t_stud_d710.sname=t_major_d710.mdean;
--为年级负责人授予查询年级视图的权利
select 'grant select on viz'||t_stud_d710.sno||' to u'||t_stud_d710.sno||';' from t_stud_d710,t_major_d710 where t_stud_d710.majorno=t_major_d710.mno and t_stud_d710.sname=t_major_d710.mdean;
