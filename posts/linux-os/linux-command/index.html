Linux 命令行& ; &&的区别

  1. command1 & command2 & command3
    三个命令同时执行

  2. command1; command2; command3
    不管前面命令执行成功没有，后面的命令继续执行

  3. command1 && command2
    只有前面命令执行成功，后面命令才继续执行

一直没太学明白的管道命令

使用 SSH 登录 Linux 实例 比如，登录我部署在腾讯云上的轻量服务器

  1. 执行以下命令，连接 Linux 云服务器。 shell ssh <username>@<IP address or domain name>
  2. username 即为前提条件中已获取的用户名，如root、ubuntu等。
  3. IP address or domain name 为您的 Linux 实例公网 IP 地址或自定义域名。
  4. 输入已获取的密码，按 Enter，即可完成登录。
